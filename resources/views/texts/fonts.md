Trip.ee uses [Sailec typeface](https://www.myfonts.com/fonts/typedynamic/sailec) for heading and body texts and [Cousine](https://fonts.google.com/specimen/Cousine) for code-related texts.

#### Default font sizes in CSS

The default sizes for heading and body texts are the following:

```
/* SomeComponent.css */

.SomeComponent__heading {
  font-size: $font-heading-xl;
}

.SomeComponent__title {
  font-size: $font-text-md;
}
```
